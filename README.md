# Booba Game

Небольшая 2D-игра на JavaScript: Буба прыгает по длинному дивану и уворачивается от "хваталок" (рук) с разными траекториями.

## Запуск

1. Откройте `index.html` в браузере.
2. Нажмите `Пробел`, чтобы начать.

## Управление

- `Пробел` / `W` / `↑`: прыжок
- `A` / `←`: влево
- `D` / `→`: вправо
- `S` / `↓`: нагнуться
- ЛКМ / тап по canvas: прыжок

## PWA

- Добавлен `manifest.webmanifest` и `service-worker.js`.
- Иконки берутся из папки `appstore-images/android`.
- Для установки откройте игру через локальный сервер (например VS Code Live Server) и выберите "Установить приложение" в браузере.

## Особенности

- Ускорение игры со временем
- Система уровней:
- 1 уровень: хваталки только справа
- 2 уровень: хваталки справа и слева
- 3 уровень: справа, слева и из глубины
- 4 уровень: повышенная скорость атак
- В конце 4 уровня появляется босс-автомат и стреляет по Бубе
- У босса есть полоска здоровья: нужно попасть по нему 5 раз
- Каждый 3-5 выстрел босс выпускает специальный синий шар
- Если синий шар приземлился на диван, Буба может пнуть его в босса
- Перед фронтальной хваталкой появляется предупреждающая мишень
- Разные типы траекторий препятствий: прямая, синус, дуга, пауза-рывок, резкий разворот назад
- Начиная с 4 уровня иногда появляется паук: запрыгивает на диван снизу с края, ползет до другого края, раньше реагирует на Бубу, замирает и прыгает на него, затем уходит спрыгиванием с края
- У Бубы 3 жизни (сердечки справа сверху)
- Проигрыш при падении с края дивана
- Локальный рекорд (`localStorage`)# boobagame
